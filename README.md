# Librariz

Librariz is a simple GUI based program to manage library. It is written in pure java and uses MySQL as database.

## Features

* This program is based on MVC architecture. 
* It uses MySQL as database.
* It uses JDBC to connect to database.
* It uses Swing for GUI.
* It uses JTable to display data.

### Book Management
    
* Add Book
* Edit Book
* Delete Book
* Search Book
* Views Book
* Views All Books
* Reserve Book


### Member Management

* Add Member
* Edit Member
* Delete Member
* Search Member
* Views Member
* Views All Members


### Staff Management

* Add Staff
* Edit Staff
* Delete Staff
* Search Staff
* Views Staff
* Views All Staffs
